Recent Projects

Automatic lights control

This was the first project that got me involved with arduino and its sensors. In this project i used 2 pir sensors which were used to detect a person and also to identify the direction of movement of the person so that it can decided whether the person is entering or leaving. And if the number of persons was more than 1 the lights would remain on. The same data was sent to blynk servers from where the user could look at the active time and number of persons.

Robotic Arm

This was for a workshop organized by the institute in this the first 2 projects were for the basic operations of the robotic arm while the third one was upto us to design a functionality for the arm. In this our team decide to make a tic tac toe playing robot In this the user entered there move and the robotic arm decides its optimal move and then the arm moved to that place on the board to place its cap on the said place. In this project i learned about the motor shield and library and also about taking input from pc and sending to arduino.

Laser Turret

I saw video about making a laser turret with servos and controlling it with mouse and i implemented the same. In that program i used visual studio and c# to take coordinates of mouse inside the window and then those were sent to arduino which then sent the same to servos and the servos moved accordingly.

Rescue

This was a project which we made for our submission in SIH. In this we made a system in which the location of the phone were sent to our own server hosted on a raspberry pi. The proposed system could be used to detect persons who are lost and do not have internet connectivty The raspberry pi was mounted to drone which then goes over the loaction and retrieves data. And this data can be used by the Rescue team for detecting the location. In this i learned about programming and using raspberry pi and interfacing it with sensors. Also in this project i used php and html to create get requests so that the phone sent the get request and the data from request was stored on database and google maps api were used to display the coordinates on website for the rescue team

Chrome Dino Automator

In this i automated the dino game in crome. Using the python libraries for keypress as well as image detection. I used pillow library to detect the obstacle and then the key was pressed. The obstacle was detected and based on its height the dinosaur jumped or ducked and i handled the transition to dark mode by taking a refrence point which made sure that the obstacle is different from background and i also handled the increasing speed of the movement
